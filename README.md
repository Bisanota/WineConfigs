# WINE CONFIGS
So, actually, in fact, is just a pre-installator to get Clip Studio Paint and FL Studio working.

In fact, files are from [this github](https://github.com/parka6060/wine-csp-patches) and some recomendations are from [this blogspot](https://offworldgirl.blogspot.com/2020/01/running-clip-studio-paint-in-wine-on.html) about Clip Studio Paint
About FL Studio, is following some instructions from [this YouTube video](https://www.youtube.com/watch?v=MNPxcxAmrhI)



# Instructions
Once system is installed [from this repository of mine](https://github.com/Bisanota/PersonalWorkStation), you get some things installed yet. Anyway, this will install every else that is not installed (and supposing isn't an arch from my files, will install all anyway).

## About CSP
Just needs a few things. Files are inside **release folder**. **Some libraries required**
Files are for activate **Timelapse** and **Export Animation**


## About FL Studio
Less tricky. **Aditional files under "fl" folder, and some libraries needed**


Thanks [parka6060](https://github.com/parka6060) for Clip Studio Paint files in Wine
